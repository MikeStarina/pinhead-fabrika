# ПИНХЭД ФАБРИКА — Сайт

Контрактное производство трикотажной одежды в Санкт-Петербурге.

## Стек
Чистый HTML5 / CSS3 / Vanilla JS. Без фреймворков и сборщиков.

## Структура файлов
- `index.html` — главная страница
- `about.html` — о фабрике
- `kontraktnoe-proizvodstvo.html` — контрактное производство
- `catalog.html` — каталог изделий
- `contacts.html` — контакты
- `zayavka.html` — форма заявки
- `poshiv-hudi.html` — пошив худи (SEO)
- `poshiv-futbolok.html` — пошив футболок (SEO)
- `coming-soon.html` — страница в разработке

## Запуск локально
Открыть `index.html` двойным кликом в браузере.

## Деплой
Загрузить все файлы в папку `/www/fabric-pnhd-admin.ru/` на хостинге reg.ru через ISPmanager.

## Дизайн-система
- `--black: #0A0A0A`
- `--blue: #1D19EA`
- `--gray: #F5F5F5`
- Шрифты: Bebas Neue / IBM Plex Mono / IBM Plex Sans
