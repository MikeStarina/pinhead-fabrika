# ПИНХЭД ФАБРИКА — Техническое описание проекта
_Последнее обновление: 23.03.2026_

---

## 1. СТЕК И ТЕХНОЛОГИИ

- **Чистый HTML5 / CSS3 / Vanilla JS** — без фреймворков
- Никаких npm, webpack, сборщиков — открывается двойным кликом
- **Шрифты:** Google Fonts (Bebas Neue, IBM Plex Mono, IBM Plex Sans)
- **Анимации:** IntersectionObserver + паттерн will-animate (fade-up при скролле)
- **Адаптив:** media queries на 1024px и 768px
- **SEO:** title, meta description, canonical, Schema.org (JSON-LD), H1/H2/H3, FAQ
- **Иконки/логотип:** встроенный SVG inline (13 path, нельзя сокращать)
